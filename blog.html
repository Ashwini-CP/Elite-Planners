What is the <template> Tag?

Posted on January 22, 2025 by Ashwini and Aruthra

The <template> tag is a special HTML element used to declare reusable content that is not rendered immediately when the page loads. Instead, it is stored in the document and can be activated and inserted into the DOM (Document Object Model) using JavaScript.

Unlike other HTML elements, content inside the <template> tag is not displayed in the browser until it is explicitly invoked. This makes it an ideal tool for building dynamic web pages where content needs to be added or updated on demand.

Here's an example of how the <template> tag works:

<template id="my-template">
  <p>This is a dynamic content block!</p>
</template>

In the example above, the content inside the <template> tag is not displayed immediately. It can be accessed and inserted into the page using JavaScript.

Why Use the <template> Tag?

The <template> tag offers several significant benefits for web developers:

  • Separation of Content and Logic: By storing reusable content in the <template> tag, you can keep your HTML structure separate from the JavaScript logic that dynamically adds or manipulates the content.
  • Improved Performance: The content inside the <template> tag is not rendered on page load, making it more efficient when dealing with large amounts of dynamic content.
  • Better Readability: It helps keep your HTML code cleaner and easier to maintain by encapsulating dynamic content in one place.
  • Ease of Use in JavaScript Frameworks: The <template> tag is frequently used in modern JavaScript frameworks (like Vue.js and Angular) to manage reusable components.

How to Use the <template> Tag in HTML

Here’s an example demonstrating how to use the <template> tag in combination with JavaScript to display dynamic content:

<template id="user-card-template">
  <div class="user-card">
    <h3><span class="name"></span></h3>
    <p>Email: <span class="email"></span></p>
  </div>
</template>

<script>
  const template = document.getElementById('user-card-template');
  const clone = document.importNode(template.content, true);

  // Modify content dynamically
  clone.querySelector('.name').textContent = 'abc';
  clone.querySelector('.email').textContent = 'abc@example.com';

  // Append the cloned content to the DOM
  document.body.appendChild(clone);
</script>

In this example, we have a template containing a user card structure. Using JavaScript, we clone the template's content, modify the text dynamically, and append the result to the DOM. This allows us to create dynamic content without reloading the page.

Practical Applications of the <template> Tag

The <template> tag is useful in a variety of scenarios:

  • Dynamic User Interfaces: It’s ideal for creating components that need to be generated on the fly, such as cards, lists, or forms, based on user interactions or fetched data.
  • Single Page Applications (SPAs): In SPAs, the <template> tag can be used to manage dynamic content that updates without a full page reload.
  • Reusable UI Elements: Developers can create templates for UI components like navigation menus, modal windows, or table rows that are reused throughout the application.
